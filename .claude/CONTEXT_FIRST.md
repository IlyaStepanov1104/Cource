## Блок 2 — Строки / Strings (60 минут)

### Часть А — Индексы и срезы (15 мин)

Строка — это **последовательность символов (sequence of characters)**.  
Каждый символ имеет индекс — положительный (слева) и отрицательный (справа).

```
 P   L   A   Y   E   R
 0   1   2   3   4   5
-6  -5  -4  -3  -2  -1
```

```python
name = "PLAYER"

name[0]       # 'P'   — первый символ
name[-1]      # 'R'   — последний, удобно
name[1:4]     # 'LAY' — от 1 включительно, до 4 не включая
name[:3]      # 'PLA' — с начала до 3
name[3:]      # 'YER' — с 3 до конца
name[::-1]    # 'REYALP' — весь в обратном порядке
name[::2]     # 'PAE' — каждый второй символ
```

#### Строки неизменяемы — Immutability

```python
name[0] = "X"        # TypeError! Строки менять нельзя

# Правильно — создаём новую строку:
name = "X" + name[1:]
```

> **Ключевой вопрос ученику:** как взять последние три символа двумя разными способами?

### Часть Б — Методы строк (20 мин)

Представим: имя игрока пришло от пользователя в «грязном» виде.

```python
raw = "  ArThUr_KiNg  "
```

**Очистка и нормализация:**

```python
raw.strip()           # убирает пробелы по краям → "ArThUr_KiNg"
raw.lower()           # всё строчными → "  arthur_king  "
raw.upper()           # всё заглавными → "  ARTHUR_KING  "
raw.capitalize()      # первая заглавная, остальные строчные
```

**Поиск и замена:**

```python
raw.replace("_", " ")    # заменяет подстроку → "  ArThUr KiNg  "
raw.find("K")            # индекс первого вхождения, -1 если нет
raw.count("r")           # сколько раз встречается
```

**Разбивка и сборка:**

```python
sentence = "fire ice thunder"
words = sentence.split()           # ['fire', 'ice', 'thunder']
" | ".join(words)                  # 'fire | ice | thunder'
"_".split(sentence)                # ← ОШИБКА: направление обратное!
"_".join(words)                    # правильно: разделитель.join(список)
```

**Проверки:**

```python
len(raw)              # длина строки
"King" in raw         # True / False
```

> **Важно:** методы возвращают **новую строку** — оригинал не меняется.

### Часть В — Итоговая функция clean_name (15 мин)

Пишем вместе. Преподаватель формулирует задачу, ученик предлагает шаги.

**Задача:** принять имя игрока, убрать лишнее, вернуть красивое имя для таблицы рекордов.

```python
def clean_name(name):
    name = name.strip()
    if len(name) == 0:
        return "Unknown"
    name = name.capitalize()
    return name

# Тесты:
print(clean_name("  aRtHuR  "))   # → Arthur
print(clean_name("   "))          # → Unknown
print(clean_name("ZARA"))         # → Zara
```
