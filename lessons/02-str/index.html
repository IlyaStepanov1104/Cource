Строки

str

Что такое строка?

Строка - это последовательность символов. 
s = "Привет"
#    0 1 2 3 4 5   - индексы
#   -6-5-4-3-2-1   - отрицательные индексы
s = 'одинарные кавычки'
s = "двойные кавычки"
s = """многострочный
текст"""
Строка в Python неизменяема - нельзя поменять один символ. Любая операция создаёт новую строку.

Операции со строками

# Сложение строк (конкатенация)
first = "Hello"
second = "World"
result = first + " " + second  # "Hello World"
# Умножение строки на число
line = "-" * 20          # "--------------------"
border = "=-" * 5 + "=" # "=-=-=-=-=-=-=-=-="

print("=" * 30)
print("Таблица рекордов")
print("=" * 30)
Умножение строки на число - удобный способ создавать разделители, рамки и паттерны.

Индексирование

s = "Python"
#    P  y  t  h  o  n
#    0  1  2  3  4  5   - индексы слева
#   -6 -5 -4 -3 -2 -1   - индексы справа

s[0]   # "P" - первый символ
s[2]   # "t"
s[-1]  # "n" - последний символ
s[-2]  # "o"
len(s)      # 6    - длина строки
"th" in s   # True - вхождение

Срезы

Общий вид: s[start : end : step] 
# Значения по умолчанию:
# start = 0      (индекс начала среза)
# end = len(s)   (индекс конца среза (не включительно))
# step = 1       (инкремент индекса)
s = "Python"

s[1:4]   # "yth"    - символы 1, 2, 3 (end не включается)
s[:3]    # "Pyt"    - start по умолчанию 0
s[3:]    # "hon"    - end по умолчанию до конца
s[::2]   # "Pto"    - каждый второй символ - 0, 2, 4
s[::-1]  # "nohtyP" - step=-1 идёт справа налево

Методы строк - регистр и пробелы

s = "  Hello, World!  "

s.upper()   # "  HELLO, WORLD!  "
s.lower()   # "  hello, world!  "
s.strip()   # "Hello, World!"  - убрать пробелы
s.lstrip()  # убрать слева
s.rstrip()  # убрать справа
name = input("Имя:").strip().lower()
# пользователь мог написать "  Иван  " или "ИВАН"

Методы строк - поиск и замена

s = "banana"

s.find("an")           # 1       - индекс первого вхождения
s.find("xyz")          # -1      - не найдено
s.count("an")          # 2       - сколько раз встречается
s.replace("an", "AN")  # "bANANa"
url = "www.python.org"

Методы строк - split и join

s = "яблоко,груша,банан"
fruits = s.split(",")    # ["яблоко", "груша", "банан"]

line = "Hello World Python"
words = line.split()     # по пробелу (по умолчанию)
items = ["яблоко", "груша", "банан"]
", ".join(items)   # "яблоко, груша, банан"
"-".join(items)    # "яблоко-груша-банан"
split - строку в список.
join - список в строку.

f-строки (f-strings)

Раньше писали так

name = "Артём"
hp = 75

# Конкатенация - неудобно
print("Игрок " + name + ": HP = " + str(hp))

# .format() - громоздко
print("Игрок {}: HP = {}".format(name, hp))

Сейчас пишут так

name = "Артём"
hp = 75

# f-строка - просто и читаемо
print(f"Игрок {name}: HP = {hp}")

# Можно писать выражения
print(f"Уровень: {hp // 10}")
f-строки читаются как обычный текст, не требуют конвертации типов и поддерживают выражения прямо внутри {}

Форматирование в f-строках

pi = 3.14159
score = 1234567

f"{pi:.2f}"       # "3.14"    - 2 знака после точки
f"{pi:.4f}"       # "3.1416"  - 4 знака после точки
f"{score:,}"      # "1,234,567" - разделитель тысяч
f"{score:_}"      # "1_234_567" - через подчёркивание
n = 42

f"{n:05d}"   # "00042" - дополнить нулями до 5 символов
f"{n:>10}"   # "        42" - выравнивание по правому краю
f"{n:<10}"   # "42        " - по левому
f"{n:^10}"   # "    42    " - по центру

Что запомнить о строках

  • Строка - последовательность символов, как список
  • Индексация и срезы работают так же, как в списках
  • Строка неизменяема - методы возвращают новую строку
  • split / join - переключение между строкой и списком
  • f-строки для красивого вывода
